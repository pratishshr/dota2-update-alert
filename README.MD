## DOTA2-UPDATE-ALERT
- Checks for new dota 2 updates from steam's RSS Feed.
- Sends mail if new update is available

### Steps

1. Install Dependencies
```bash
 $ npm install
```

2. Rename config.json.template to config.json and add required keys

3. Run script
```bash
$ npm start
```

4. Run and watch script 
```bash
$ npm run dev
```
