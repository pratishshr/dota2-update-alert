## DOTA2-UPDATE-ALERT
- Checks for new dota 2 updates from steam's RSS Feed.
- Sends mail if new update is available

### Steps

- **Install Dependencies**
```bash
 $ npm install
```

- **Rename config.json.template to config.json and add required keys**

- **Run script**
```bash
$ npm start
```

- **Run and watch script**
```bash
$ npm run dev
```
